# kalmbuddha - Интересные места Элисты (MVP с регистрацией)

Минимальный рабочий проект: **React (Vite) + Node.js/Express + PostgreSQL (Sequelize) + JWT**.

## Быстрый старт (локально)

### 1) Требования
- Node.js 18+ (или 20+)
- PostgreSQL 14+ (локально установленный)

### 2) Настройка переменных окружения (server)
Скопируй файл `server/env.example` в `server/.env` и поправь значения под себя.

### 4) Установка зависимостей
В корне проекта:

```bash
npm --prefix server i
npm --prefix client i
```

### 5) Запуск
В двух окнах терминала:

```bash
npm --prefix server run dev
```

```bash
	npm --prefix client run dev
```
