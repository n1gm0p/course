# 📐 Руководство по дизайну проекта "Элиста места"

Это подробное руководство поможет вам понять, где находится код дизайна и как его изменить.

---

## 📁 Структура проекта

```
kalmbuddha/
├── client/                    # Фронтенд (React + TypeScript)
│   ├── public/               # Статические файлы (изображения, иконки)
│   │   └── images/          # Изображения сайта
│   │       ├── elista-logo.svg    # Логотип сайта
│   │       ├── elista-hero.jpg   # Фоновое изображение главной страницы
│   │       └── elista-temple.jpg # Изображение храма
│   ├── src/                  # Исходный код приложения
│   │   ├── pages/           # Страницы сайта
│   │   ├── App.tsx         # Главный компонент с навигацией
│   │   ├── main.tsx         # Точка входа приложения
│   │   └── style.css        # ВСЕ СТИЛИ ПРОЕКТА (главный файл дизайна!)
│   └── index.html           # HTML-шаблон страницы
└── server/                   # Бэкенд (Node.js + Express)
```

## 🎨 ГЛАВНЫЙ ФАЙЛ ДИЗАЙНА: `client/src/style.css`

**Путь:** `client/src/style.css`

**Что делает:** Это ЕДИНСТВЕННЫЙ файл со всеми стилями сайта. Здесь находятся:
- Цвета
- Шрифты
- Размеры элементов
- Отступы и промежутки
- Анимации
- Адаптивный дизайн (для мобильных)

**Как изменить дизайн:**
1. Откройте файл `client/src/style.css`
2. Найдите нужный класс (например, `.navbar` для навигации)
3. Измените значения свойств (цвет, размер, отступы)

## 🏗️ ОСНОВНЫЕ КОМПОНЕНТЫ И ИХ СТИЛИ

### 1. **Навигация (Navbar)**

**Где находится код:**
- **Компонент:** `client/src/App.tsx` (строки 16-47)
- **Стили:** `client/src/style.css` (строки 36-68)

**Что делает:**

**Основные классы стилей:**
```css
.navbar          /* Контейнер навигации */

.nav-right       /* Правая часть (кнопки входа) */
```

**Как изменить:**
- **Цвет фона навигации:** `client/src/style.css`, строка 41: `background: rgba(255, 255, 255, 0.9);`
- **Размер логотипа:** `client/src/style.css`, строка 64: `height: 32px;`
- **Цвет логотипа (фильтр):** `client/src/style.css`, строка 67: `filter: invert(1) brightness(0.2);`

### 5. **Цветовая палитра**

**Где находится:** `client/src/style.css`


### 7. **Адаптивный дизайн (Мобильные устройства)**

**Где находится:** `client/src/style.css` (строки 315-393)

**Что делает:**
- Меняет дизайн для экранов меньше 900px и 640px
- Делает навигацию вертикальной на мобильных
- Уменьшает размеры элементов

**Основные медиа-запросы:**
```css
@media (max-width: 900px) {  /* Планшеты */
  /* Стили для планшетов */
}

@media (max-width: 640px) {  /* Мобильные телефоны */
  /* Стили для телефонов */
}
```

**Как изменить:**
- Откройте `client/src/style.css`
- Найдите нужный медиа-запрос (строка 317 или 346)
- Измените стили внутри блока `@media`

## 📱 АДАПТИВНОСТЬ

Проект автоматически адаптируется под разные размеры экранов:

- **Большие экраны (>900px):** Полный дизайн
- **Планшеты (640px-900px):** Уменьшенные отступы, адаптированная сетка
- **Мобильные (<640px):** Вертикальная навигация, одна колонка карточек

Все адаптивные стили находятся в `client/src/style.css` в блоках `@media`.
